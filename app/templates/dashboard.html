基于 Word2Vec 的情感分析系统

利用深度学习与自然语言处理技术，对文本进行精准情感倾向判断

立即体验

我们的系统可以做什么？

支持单条/批量分析，适用于评论监控、舆情分析、用户反馈整理等场景

高精度情感识别

通过 Word2Vec 向量建模 + SVM/XGBoost 模型，实现对正面、负面、中性情绪的准确判断。

灵活的模型选择

支持多种机器学习模型切换，可实时查看不同模型的效果对比与置信度。

历史记录追踪

自动保存每一次分析结果，支持查看、搜索、导出等功能，便于后续数据复用与分析。

系统架构

基于 Flask 框架构建，后端提供 RESTful API，前端采用 Bootstrap 美化界面

  • 后端：  Flask + SQLAlchemy + Flask-Login
  • 分析引擎：  SVM / XGBoost / Word2Vec
  • 数据库：  MySQL 存储分析记录与用户信息
  • 前端：  HTML + Bootstrap 5 + Chart.js 可视化

如何使用

三步完成情感分析，轻松上手

输入文本

在“单文本分析”页面输入你想分析的语句，如商品评价、社交媒体内容等。

选择模型

可以选择不同的算法模型（SVM/XGBoost），并查看其预测结果与置信度。

获取结果

系统返回情绪分类结果，并以图表形式展示整体趋势与分布情况。

关于本系统

这是一个基于 Python Flask 和 NLP 技术构建的情感分析平台，可用于电商评论、社交媒体、问卷调查等多种场景下的文本分析任务。
